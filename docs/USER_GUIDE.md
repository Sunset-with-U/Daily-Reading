## 1. 安装与首次启动

1. 到仓库 **[Releases](../../../releases)** 页下载最新的 `Daily Reading-x.x.x.dmg`。
2. 双击 `.dmg`，把 **Daily Reading** 图标拖进「应用程序」文件夹。
3. **第一次打开：右键点击图标 → 打开 → 再点「打开」。**
   开源应用没有付费的 Apple 开发者签名，macOS 首次会拦截；右键打开只需做一次，
   以后正常双击即可。（备选：终端执行
   `xattr -dr com.apple.quarantine "/Applications/Daily Reading.app"`）
4. 启动后：
   - 屏幕顶部菜单栏出现 **DR** 图标——App 常驻这里，**关掉窗口不会退出**；
   - 首次没有数据时，窗口会自动带你进入「设置」页。

菜单栏 DR 图标的菜单：**打开看板 / 立即运行早报 / 立即运行晚报 / 退出 Daily Reading**。

## 2. 三分钟完成初始配置

只需两步就能跑起来：

**第一步：粘一个 AI Key。** 「设置 → API 密钥」区，把你申请的 Key 粘进对应供应商的框里，点「保存」，左侧圆点变绿即成功。四家任选其一：

| 供应商 | 申请地址 | 特点 |
|---|---|---|
| Claude（默认推荐） | console.anthropic.com | 分析质量最佳，支持省钱 Batch |
| ChatGPT（OpenAI） | platform.openai.com | 支持省钱 Batch |
| Gemini（Google） | aistudio.google.com | 有免费额度 |
| DeepSeek | platform.deepseek.com | 最便宜，中文强 |

**第二步：点一次「立即运行」。** 「设置 → 运行与定时」区，班次留「自动判断」，点 **立即运行**。首次全量抓取约 3–10 分钟；AI 分析实时模式再等十几分钟、省钱 Batch 模式约 30–75 分钟。完成后弹系统通知，点开窗口即可阅读。

## 3. 设置面板详解

顶部导航最右侧「设置」，四个分区，所有改动即存即生效（写入本机用户目录，出厂配置永不被破坏）：

### 3.1 AI 引擎

| 项 | 说明 |
|---|---|
| AI 供应商 | Claude / ChatGPT / Gemini / DeepSeek 随时切换；切换后请确认对应 Key 已配置 |
| 调用方式 | **实时**＝逐条即时出结果；**省钱 Batch**＝五折价格、等 30–75 分钟（Claude/ChatGPT 支持；Gemini/DeepSeek 选了也会自动按实时执行并提示） |
| 逐条分析模型 / 每日报告模型 | 每家供应商可分别指定两个模型；留默认即可，想省钱可把报告模型降档 |
| 每日 AI 条数上限 | 成本总闸（默认 600 条/天）；调低直接省钱 |

### 3.2 API 密钥

- 四家 AI Key + 两个可选数据源 Key（twitterapi.io、FRED）。
- 全部保存进 **macOS 钥匙串**（「钥匙串访问」App 里搜 *Daily-Reading* 可见），
  界面永不回显明文；输入新值点保存即替换，点删除即清除。
- **twitterapi.io**（可选，约 $15–25/月）：配置后自动启用 X（Twitter）的 104 个精选账号组；不配则用免费 Telegram 快讯层顶替。
- **FRED**（免费注册）：市场快照里的宏观数据面板需要它。

### 3.3 信息源

- **RSSHub 实例**：财联社电报、财新等 8 个中文源需要。填公共实例 `https://rsshub.app` 或自建地址；不填则这些源自动跳过，其余照常。
- **源列表**：120+ 源按分类折叠展示，每行一个拨杆开关——关掉即下轮运行不再抓取；顶部输入框可按名称/id 筛选。
- **添加自定义源**：底部填「名称 + RSS 地址」点添加，保存后并入你的信息流，享受同样的 AI 分析。
- 改完记得点「**保存信息源设置**」；配置非法（如缺 URL）会被直接拒绝并提示原因，不会写坏配置。

### 3.4 运行与定时

- **手动运行**：任意时刻补跑一班；班次可强制指定早报/晚报。
- **早报/晚报时间**：默认北京 07:00 / 20:00，**每个人都可以改成自己想要的整点**（0–23）。保存后下一次检查即生效，无需重启。
- 面板底部显示下次自动运行时间。

## 4. 七个视图怎么用

| 视图 | 用途 |
|---|---|
| **今日报告** | 当日深度研报：一句话头条 → TL;DR 要点 → 全球叙事 → 跨资产联动 → 分市场深析 → 风险与日历 → 情绪面。早报/晚报可切换 |
| **信息流** | 全部条目卡片流。顶部可按重要性（S/A/B/C）、市场、来源筛选 + 关键词搜索；S/A 级条目点开看影响推演（方向/资产/时间尺度/置信度） |
| **关注清单** | AI 从当日信息中提出的交易关注标的（个股/ETF/期货/期权/外汇/加密），附方向、论述、关键价位与 TradingView 小图 |
| **市场快照** | 宏观利率、波动率期限结构、持仓、贵金属沪伦溢价、加密衍生品、情绪指标、财经日历、自选清单技术指标 |
| **归档** | 按日期回看任意一天的报告与数据；顶栏日期切换器也可直接翻页 |
| **源状态** | 每个信息源本轮抓取的成败、条数、延迟——判断"某个源是不是坏了"看这里 |
| **设置** | 见上一节 |

## 5. 自动运行与推送时间

- 到点自动执行完整流程：抓取 → 去重 → 市场快照 → AI 逐条分析 → 深度报告 → 系统通知推送。
- **错过补跑**：白天电脑没开机？晚上打开 App，会自动补跑最近错过的一班（只补最近一班，不重复花早班的钱）。
- 同一班次当天只跑一次；手动跑过后定时器不会再重复。
- 想换节奏：在「设置 → 运行与定时」改两个整点即可，比如改成 6 点和 22 点。

## 6. 白天 / 黑夜模式

窗口右上角 **☾ / ☀** 按钮一键切换。首次启动跟随系统外观；手动切换后记住你的选择。全部视图（含 TradingView 图表）同步换肤。

## 7. 花费控制

从大到小的三个闸门：

1. **每日 AI 条数上限**（设置 → AI 引擎）：600 → 300，费用即减半；
2. **调用方式选省钱 Batch**（Claude/ChatGPT）：同样的量五折；
3. **换更便宜的模型/供应商**：报告模型降一档、或整体切到 DeepSeek，可以压到每天几毛钱。

参考：默认 Claude + Batch + 600 条 ≈ $1.5–2/天。各家控制台可实时核对账单，建议设置月度消费上限。

## 8. 数据与隐私

- 一切都在本机：数据与配置在 `~/Library/Application Support/Daily-Reading/`，密钥在 macOS 钥匙串。
- 网络请求只有两类：直连各信息源站点抓内容、直连你所选 AI 供应商的官方 API 送分析。没有任何中间服务器，不上传任何数据。
- 本机只保留每条 ≤500 字的工作摘录，历史数据 180 天自动瘦身。
- 完全重置：删除上述目录（+ 钥匙串中的 Daily-Reading 条目），重新打开 App 即回到出厂状态。

## 9. 故障排查

| 症状 | 处理 |
|---|---|
| 打不开，提示"无法验证开发者" | 右键 → 打开（见第 1 节），只需一次 |
| 运行完成但条目没有 AI 分析 | 设置里确认所选供应商的密钥圆点是绿的；没配 Key 时管线照常抓取归档、只跳过 AI，补配后下班次自动分析 |
| 某个源一直红（失败） | 上百个源总有几个在改版，单源失败不影响整体；「源状态」看具体报错，长期失败可在设置里关掉它 |
| 财联社等中文快讯源是空的 | 需要配置 RSSHub 实例（设置 → 信息源） |
| Batch 模式等了很久没结果 | Batch 由供应商排队（最长 75 分钟死线）；超时的部分自动登记，下一班开头自动回收合并，不丢也不重复付费 |
| 窗口关了找不到 App | 它还活着——看屏幕顶部菜单栏的 **DR** 图标 → 打开看板 |
| 想彻底退出 | 菜单栏 DR → 退出 Daily Reading |
